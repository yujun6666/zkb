# ZhongKong_Cup_2026_MAE_Transformer
2026年“中控杯”智能制造挑战赛 - 赛题1：时序大模型驱动的流程工业智能预测与控制优化

## 📌 项目简介
本项目针对流程工业工厂工段真实脱敏数据集（包含65个工艺监测指标），构建了一套基于 **MAE (Masked Autoencoders) + Transformer 块** 的高精时序预测控制大模型。
为冲刺极限 MSE，本项目分为 V1（稳健进阶版）与 V2（激进冲榜版），针对 65 特征的刚性要求进行了精细化对齐与底层重构。

## 🛠️ 核心技术特性
- **双模并行架构**：
  - **[V1版]** 采用深度多层 `Block` 空间/时域特征解耦，结合余弦退火学习率（Cosine Annealing）与早停机制（Early Stopping）。
  - **[V2版]** 引入前沿学术突破 **Xicor Attention**，使用 Chatterjee 秩相关系数的 L1 距离惩罚项替换传统点乘，暴力捕捉化工变量间的非线性耦合关系。
- **自回归滚动推理**：采用 24 步轻量级滚动预测外推技术，连续滑动外推 500 次，完美凑出初赛要求的未来 12,000 步长序列，彻底绝杀 18 亿参数带来的内存爆炸隐患。
- **工业级数据解毒**：摒弃粗暴填零，采用 `interpolate` 双向线性插值与极值剔除，最大程度保留真实物理特征的时序连贯性。

## 📊 数据集与交卷形态
- **输入**：60万行原始时序长河，自动切片为 10 个 Batch。
- **输出**：生成符合官方平台要求的 `.npy` 文件，最终几何形状严格咬合 `(3, 10, 12000)`。

## 🗂️ 文件树
- `perfect_pipeline_v1_advanced.py` : V1 稳健进阶版（标准 Transformer + 早停）
- `perfect_pipeline_v2_xicor.py` : V2 激进冲榜版（Xicor Attention 非线性引擎）
- `plot_future_curve.py` : 智能自适应预测轨迹可视化画图工具
- `view_as_csv.py` : 历史与未来无缝拼接表格生成器
- `export_raw_to_csv.py` : 原始 NPY 全量数据导出工具
