# ZhongKong_Cup_2026_MAE_Transformer
2026年“中控杯”智能制造挑战赛 - 赛题1：时序大模型驱动的流程工业智能预测与控制优化

## 📌 项目简介
本项目针对流程工业工厂工段真实脱敏数据集（包含65个工艺监测指标），构建了一套基于 **MAE (Masked Autoencoders) + Transformer 块** 的高精时序预测控制大模型。
项目完美还原了学长论文级的深度全量自注意力网络架构，并针对今年 65 特征的刚性要求进行了多轮几何对齐与性能优化。

## 🛠️ 核心技术特性
- **高级模型架构**：全量集成 `SkeleEmbed`、`Attention` 以及多层 `Block` 空间/时域特征解耦模块。
- **自回归滚动推理**：采用 24 步轻量级滚动预测外推技术，连续滑动外推 500 次，完美凑出初赛要求的未来 12,000 步长序列，彻底绝杀 18 亿参数带来的内存爆炸隐患。
- **全自动数据解毒**：内置 `nan_to_num` 物理清洗机制与梯度裁剪（Clip Grad），彻底压制原始数据中潜伏的 NaN 病毒，保障数值稳定不崩溃。

## 📊 数据集与交卷形态
- **输入**：60万行原始时序长河，自动切片为 10 个 Batch。
- **输出**：生成符合官方平台要求的 `predictions_2026_submit.npy` 文件，最终几何形状严格咬合 `(3, 10, 12000)`。

## 🗂️ 文件树
- `perfect_pipeline_2026.py` : 包含预训练、微调特训、滚动推理的一体化通关主脚本
- `check_npy.py` : 本地二进制预测结果矩阵完整性校验工具
- `view_as_csv.py` : 历史已知与预测未来的无缝拼接表格生成器
